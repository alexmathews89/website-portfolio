A few other frameworks and technologies used in this app are Puppeteer, jest, Atatus and this project also uses a little bit of object oriented programming to create certain app alerts. The Chat App is a project developed with React Native. With this application a user can enter their name and choose a color for their chat screen before entering a chat room where they can also take and send a photo, choose a photo from their library to send as well as send their location. The Chat App was developed for a mobile device, so it isn't hosted on the web but the app can be ran by simply creating an Expo account and then downloading and installing Expo Go on your device(smartphone) and/or a virtual Android emulator and then downloading the GitHub project folder here. From there, navigate to the project folder using the terminal and run the command "npm start" or "expo start." You can then open Expo Go on your device and select the chat app to create a username and enter the chat room. The technologies being used in this project are React Native, Firestore, Expo, GiftedChat, AsyncStorage, StackNavigator, expo-location, expo-image-picker and react-native-maps.
